# customer-churn-prediction
ML project for predicting customer churn using Python

This project predicts customer churn using machine learning models.

## Dataset
Used a telecom customer churn dataset (public dataset).

## Tools
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Steps
1. Data cleaning & preprocessing  
2. Feature engineering  
3. Model training & evaluation  
4. Prediction & performance analysis  

## Results
- Model accuracy: ~80% (example)
- Best model: Random Forest
